## Skills and Competencies Requested During Onboarding

### Communication & Collaboration Tools

| Skill | Details |
|-------|---------|
| **Slack** | Configure notifications, use threads, tag people, manage channels, integrate with Notion |
| **Google Meet** | Video conferencing, camera-on policy, call-in for connectivity issues |
| **Google Workspace (Gmail, Docs, Sheets, Slides)** | Email, document drafting, collaboration |
| **Miro** | Digital whiteboarding, brainstorming |

### AI & LLM Proficiency

| Skill | Details |
|-------|---------|
| **Gemini (Google)** | Approved GenAI tool for drafting, summarizing, brainstorming, code assistance |
| **AI data security awareness** | Know what NOT to input (PII/PHI, credentials, API keys, IP) |

### Development & Technical Tools

| Skill | Details |
|-------|---------|
| **GitHub** | Code management, repository access, version control |
| **Databricks** | Data engineering, data science, ML tasks |
| **Azure DevOps** | Used for some client projects (work items, code storage) |

### Project Management & Process

| Skill | Details |
|-------|---------|
| **Notion** | Project databases, work items, artifacts, risks, stakeholder tracking |
| **Everhour** | Daily time tracking, time rounding setup, monthly report generation |
| **Agile methodology** | Sprints, sprint goals, Good-Better-Best planning, Kanban/Scrum |
| **The Synaptiq Way** | Internal delivery methodology (Initiation → Execution → Review → Maintenance) |

### Compliance & Security

| Skill | Details |
|-------|---------|
| **HIPAA compliance** | Mandatory training within first 2 weeks (KnowBe4) |
| **Phishing awareness** | KnowBe4 phishing alert button in Gmail |
| **Client data handling** | IP/confidentiality policies, proper document storage (no local drives) |

### Administrative

| Skill | Details |
|-------|---------|
| **Invoicing** | Monthly report creation, invoice templates, PDF naming conventions |
| **JustWorks** | Payroll/benefits setup (employees) or contractor payment setup |
| **Email signature setup** | Synaptiq-branded Gmail signature |

### Soft Skills / Cultural Competencies ("Synaptitudes")

The onboarding deck references 10 cultural principles employees are expected to embody:

1. Relentless Pursuit of Client Impact
2. Bias for Action
3. Embrace Constraints & Ambiguity
4. Say the Hard Thing Early
5. Have Ownership Mindset
6. Achieve More with Less
7. Be Humble, Be Sharp
8. All In for the Team
9. Coordinate by Sharing Publicly, Early, & Often
10. Learn & Teach Persistently

## Key Takeaways for Managers

### Potential Training Gaps to Watch For

- **Gemini/AI literacy** — New hires may default to ChatGPT/Claude; need redirection to approved tools.
- **Agile fluency** — Sprint planning, Good-Better-Best framework is Synaptiq-specific.
- **Notion + Everhour proficiency** — Critical for daily operations and getting paid.
- **HIPAA/data security** — Mandatory and time-bound (2-week deadline).
- **Client communication etiquette** — Internal vs. external Slack channels, coordinating with Tech Lead/PM before client-facing messages.
